Question And Answer

  • 1. What's different between map,find,filter and forEach?

    map: The map method used for creating new array by applying a given a function.

    find: The find method returns the first element in the array.

    filter: The filter method creates a new array with all the elements that pass the test implemented by the provided function.

    forEach: The forEach method executes a provided function once for each array element. It does not return a new array.

  • 2. What's different between Local Storage, Session Storage, and Cookies?

    Local Storage: Local Storage API allows you to store key-value pairs in the browser.

    Session Storage: Session Storage API allows you to store key-value pairs in the browser that are only available for the duration of the current session

    Cookies: Cookies are small text files that are stored on the user's computer by a website.

  • 3. What is even loop and how it does work in js?

    The event loop in JavaScript is a mechanism that manages the execution of asynchronous code by processing a queue of events or messages, and executing their associated callback functions when the call stack is empty.
    Event loop continuously monitors a queue of events or messages and executes their corresponding callback functions when the call stack is empty, allowing JavaScript to handle asynchronous operations efficiently without blocking the execution of other code.
